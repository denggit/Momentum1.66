### 1.2 核心设计原则

1. **零内存分配热路径**：策略计算线程禁止调用`malloc`/`new`，全部使用预分配对象池
2. **无锁数据传递**：WebSocket线程→策略线程使用单生产者单消费者(SPSC)无锁环形缓冲区
3. **事件驱动状态机**：状态转换由Tick事件触发，**严禁**使用定时器驱动
4. **单向数据流**：策略层只发命令，不等待执行结果，执行状态由独立监控线程跟踪

## 3. 核心算法实现

### 3.1 Range Bar生成器（Algorithm 1）

**输入**: `NormalizedTick`流  
**输出**: `RangeBar`流  
**触发条件**: 价格物理位移达到`tick_range` (20 Ticks)

```cpp
class RangeBarGenerator {
private:
    RangeBar current_bar;
    double open_px_base;          // 当前Bar的基准价
    std::queue<RangeBar> history; // 历史Bar缓存（用于CVD计算）
    
public:
    void OnTick(const NormalizedTick& tick) {
        if (!current_bar.is_initialized) {
            // 初始化新Bar
            current_bar.open_ts = tick.ts;
            current_bar.open_px = tick.px;
            current_bar.high_px = tick.px;
            current_bar.low_px = tick.px;
            open_px_base = tick.px;
            current_bar.total_buy_vol = (tick.side == 1) ? tick.sz : 0;
            current_bar.total_sell_vol = (tick.side == -1) ? tick.sz : 0;
            current_bar.tick_count = 1;
            current_bar.is_closed = false;
            return;
        }
        
        // 更新当前Bar
        current_bar.high_px = std::max(current_bar.high_px, tick.px);
        current_bar.low_px = std::min(current_bar.low_px, tick.px);
        current_bar.close_px = tick.px;
        
        if (tick.side == 1) current_bar.total_buy_vol += tick.sz;
        else current_bar.total_sell_vol += tick.sz;
        
        current_bar.delta = current_bar.total_buy_vol - current_bar.total_sell_vol;
        current_bar.tick_count++;
        
        // 检查是否触发闭合条件
        double displacement = std::abs(tick.px - open_px_base) / tick_size;
        if (displacement >= tick_range) {
            CloseBarAndEmit(tick);  // 闭合当前Bar，溢出Tick带入下根
        }
    }
    
private:
    void CloseBarAndEmit(const NormalizedTick& overflow_tick) {
        current_bar.is_closed = true;
        EmitToKDEEngine(current_bar);
        
        // 新开Bar，处理溢出
        ResetWithOverflow(overflow_tick);
    }
};
```

### 3.2 KDE与LVN提取引擎（Algorithm 2）

**输入**: `RangeBar`序列（脉冲波段）  
**输出**: `LVN_Zone`数组（最多3个区域）

```cpp
class KDEEngine {
public:
    std::vector<std::pair<double, double>> ExtractLVN(const ImpulseSlice& slice) {
        // Step 1: 稳健Silverman带宽计算
        double h = CalculateRobustBandwidth(slice.ticks);
        
        // Step 2: 构建加权KDE
        std::map<double, double> density_map;
        for (const auto& tick : slice.ticks) {
            for (double px = tick.px - 5*tick_size; px <= tick.px + 5*tick_size; px += tick_size) {
                double k = GaussianKernel((tick.px - px) / h);
                density_map[px] += tick.sz * k;
            }
        }
        
        // Step 3: 寻找局部极小值（LVN）
        auto minima = FindLocalMinima(density_map);
        
        // Step 4: 密度阈值过滤（<30%分位）
        double avg_density = CalculateAverage(density_map);
        std::vector<std::pair<double, double>> lvn_zones;
        
        for (const auto& min_px : minima) {
            if (density_map[min_px] < avg_density * 0.3) {
                lvn_zones.push_back({min_px - tick_size, min_px + tick_size});
            }
        }
        
        // 最多返回3个，按密度升序排序
        std::sort(lvn_zones.begin(), lvn_zones.end(), 
                  [](auto& a, auto& b) { return a.first < b.first; });
        if (lvn_zones.size() > 3) lvn_zones.resize(3);
        
        return lvn_zones;
    }
    
private:
    double CalculateRobustBandwidth(const std::vector<NormalizedTick>& ticks) {
        // Silverman稳健法则: h = 0.9 * min(std, IQR/1.34) * n^(-1/5)
        std::vector<double> prices;
        for (const auto& t : ticks) prices.push_back(t.px);
        
        double std_dev = StandardDeviation(prices);
        double iqr = Percentile(prices, 75) - Percentile(prices, 25);
        double n = ticks.size();
        
        return 0.9 * std::min(std_dev, iqr / 1.34) * std::pow(n, -0.2);
    }
};
```

### 3.3 Triple-A状态机（Algorithm 3 - 核心）

**状态定义**:
```cpp
enum class TripleAState {
    IDLE,                       // 休眠：价格不在任何LVN区域内
    MONITORING_ABSORPTION,      // 监测中：价格进入LVN，监测努力vs结果
    ABSORPTION_CONFIRMED,       // 吸收确认：检测到高CVD低位移
    ACCUMULATING,               // 积累中：波动压缩+Tick密度上升
    AGGRESSION_TRIGGERED        // 攻击触发：满足所有开仓条件
};
```

**状态转换图**:

```
                    [价格进入LVN区域]
    IDLE ──────────────────────────────────────► MONITORING_ABSORPTION
         ◄──────────────────────────────────────┘ [价格离开LVN区域]
                          (reset)
    
    MONITORING_ABSORPTION ──[ effort>2σ && |result|<2ticks ]──► ABSORPTION_CONFIRMED
                                    │
                                    │ [波动>3ticks 或 超时2min]
                                    ▼
                              ACCUMULATING ◄──────┘
                                    │
                                    │ [Tick数>200 && 波动<3ticks]
                                    ▼
                              AGGRESSION_TRIGGERED ──[大单气泡/失衡/CVD斜率]──► EXECUTE_LONG/SHORT
```

**详细判定逻辑**:

```cpp
class TripleAStateMachine {
public:
    void OnNormalizedTick(const NormalizedTick& tick) {
        UpdateContext(tick);
        
        switch (ctx.state) {
            case TripleAState::IDLE:
                if (IsPriceInLVN(tick.px)) {
                    TransitionTo(MONITORING_ABSORPTION, tick);
                }
                break;
                
            case TripleAState::MONITORING_ABSORPTION:
                // 累积努力与结果数据
                ctx.effort_cvd += tick.side * tick.sz;
                ctx.result_delta_px = tick.px - ctx.entry_px;
                ctx.tick_count_in_lvn++;
                
                // 检查是否满足吸收条件
                if (ctx.tick_count_in_lvn >= absorption.window_ticks) {
                    double cvd_mean = GetHistoricalCVDMean(absorption.window_ticks);
                    double cvd_std = GetHistoricalCVDStd(absorption.window_ticks);
                    double zscore = (ctx.effort_cvd - cvd_mean) / cvd_std;
                    
                    bool effort_ok = std::abs(zscore) > absorption.effort_zscore_threshold;
                    bool result_ok = std::abs(ctx.result_delta_px) <= absorption.max_result_px_deviation;
                    bool efficiency_ok = std::abs(ctx.result_delta_px / ctx.effort_cvd) > absorption.cvd_efficiency_min;
                    
                    if (effort_ok && result_ok && efficiency_ok) {
                        TransitionTo(ABSORPTION_CONFIRMED, tick);
                    } else if (!IsPriceInLVN(tick.px)) {
                        TransitionTo(IDLE, tick); // 价格离开LVN，重置
                    }
                }
                break;
                
            case TripleAState::ABSORPTION_CONFIRMED:
                // 监测积累条件
                ctx.acc_tick_count++;
                UpdateAccumulationVolatility(tick.px);
                
                bool range_ok = (GetCurrentRangeTicks() <= accumulation.max_range_ticks);
                bool density_ok = (ctx.acc_tick_count >= accumulation.min_tick_count);
                bool timeout = (Now() - ctx.entry_ts > accumulation.timeout_ms * 1000000);
                
                if (range_ok && density_ok) {
                    TransitionTo(ACCUMULATING, tick);
                } else if (GetCurrentRangeTicks() > accumulation.max_range_ticks || timeout) {
                    TransitionTo(IDLE, tick); // 区间过大或超时，重置
                }
                break;
                
            case TripleAState::ACCUMULATING:
                // 监测攻击信号（最高频，微秒级响应）
                if (DetectAggression(tick)) {
                    ctx.trigger_px = tick.px;
                    ctx.aggression_sz = CalculateBubbleSize(tick);
                    TransitionTo(AGGRESSION_TRIGGERED, tick);
                } else if (!IsPriceInLVN(tick.px)) {
                    TransitionTo(IDLE, tick);
                }
                break;
                
            case TripleAState::AGGRESSION_TRIGGERED:
                // 已发送交易指令，等待执行层确认
                // 此状态下不再处理新Tick，由RiskManager接管
                break;
        }
    }
    
private:
    bool DetectAggression(const NormalizedTick& tick) {
        // 并行检测三个条件：
        
        // 1. 大单气泡检测（动态阈值）
        double bubble_threshold = GetPercentileVolume(aggression.print_percentile);
        bool bubble_ok = (tick.sz > bubble_threshold);
        
        // 2. 足迹失衡检测（3档连续300%失衡）
        bool imbalance_ok = CheckFootprintImbalance(tick, aggression.imbalance_ratio, aggression.stacked_levels);
        
        // 3. CVD斜率确认
        double current_slope = (ctx.effort_cvd - last_cvd) / (tick.ts - last_ts) * 1000000; // 每微秒
        bool slope_ok = std::abs(current_slope) > aggression.cvd_slope_threshold;
        
        // 方向一致性检查（与Layer1宏观状态一致）
        bool direction_ok = (tick.side == GetMacroDirection()); // +1 for long, -1 for short
        
        return (bubble_ok || imbalance_ok) && slope_ok && direction_ok;
    }
};
```

## 4. 执行与风控层（Layer 4）

### 4.1 动态仓位计算

```cpp
struct PositionSizingResult {
    double qty;           // 合约数量
    double stop_px;       // 止损价格（物理锚点）
    double breakeven_px;  // 保本价格
};

PositionSizingResult CalculatePosition(double entry_px, double trigger_px, double equity) {
    PositionSizingResult result;
    
    // 风险金额 = equity * 0.25%
    double risk_amount = equity * risk_management.risk_per_trade_pct;
    
    // 止损距离 = |entry_px - (trigger_px - 1*tick_size)| 对于多单
    double stop_distance = std::abs(entry_px - (trigger_px - risk_management.stop_buffer_ticks * tick_size));
    
    // 合约数量 = risk_amount / stop_distance
    result.qty = std::floor(risk_amount / stop_distance * 100) / 100; // 精确到2位小数
    
    // 止损价格 = trigger_px - 1*tick_size (多单)
    result.stop_px = trigger_px - risk_management.stop_buffer_ticks * tick_size;
    
    // 保本价格 = entry_px + 1*tick_size (覆盖手续费)
    result.breakeven_px = entry_px + risk_management.breakeven_trigger_ticks * tick_size;
    
    return result;
}
```

class CVDTrailingGuard {
public:
    void OnPositionOpened(double entry_px, int side, double qty) {
        this->entry_px = entry_px;
        this->side = side;
        this->qty = qty;
        this->highest_cvd_since_entry = GetCurrentCVD();
    }
    
    void OnTick(const NormalizedTick& tick) {
        if (side == 1 && tick.px > entry_px) { // 多单浮盈中
            double current_cvd = GetCurrentCVD();
            highest_cvd_since_entry = std::max(highest_cvd_since_entry, current_cvd);
            
            // 检测CVD背离：价格新高但CVD未新高
            if (tick.px > highest_px_since_entry * 1.002 && current_cvd < highest_cvd_since_entry * 0.995) {
                // 触发保本：将止损移至breakeven
                AmendStopLossToBreakeven();
            }
        }
    }
};

## 7. 常见工程陷阱与强制约束

| 陷阱 | 强制解决方案 |
|------|-------------|
| **定时器驱动** | 严禁使用`setInterval`/`sleep`检查状态，必须用Tick事件触发状态转换 |
| **固定时间窗** | 吸收/积累/攻击三阶段时间尺度不同，禁止用固定15秒窗口包裹 |
| **孤立CVD** | 必须同时检查价格位移，计算`effort/result`效率比 |
| **固定止损** | 止损必须锚定在`trigger_px`后方1-2 Tick，禁止固定点数 |
| **限价单抢跑** | 攻击阶段必须使用市价单或IOC，禁止挂限价单等待成交 |
| **内存分配** | 热路径禁止`malloc`，使用预分配对象池 |
| **锁竞争** | WebSocket线程与策略线程零共享，仅用原子操作 |
