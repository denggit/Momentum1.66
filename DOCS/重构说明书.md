Momentum1.66 三号引擎重构计划

上下文与目标

三号引擎（engine_3_orderflow）是当前实盘运行的核心引擎，但存在以下问题：

1. 硬编码泛滥：参数（杠杆50倍、仓位80%、500万美元阈值等）直接写死在代码中

2. 状态共享危险：通过self.trader.of_wall_price直接修改变量进行模块间通信

3. 职责不纯：

- Engine3Commander类承担了太多职责（策略协调、状态管理、邮件通知）

- OKXTrader类包含复杂的4阶段止损跟踪逻辑

4. 配置缺失：config/orderflow目录为空，没有币种特定配置文件

5. 架构耦合：缺乏单向数据流设计，模块间存在反向调用

重构目标：

- 保持现有功能逻辑不变

- 实现配置驱动，支持多币种（ETH, BTC, SOL, DOGE等）

- 解耦模块，建立单向数据流：DataFeed → Strategy → Risk → Execution

- 消除危险的状态共享方式

- 分离交易执行与止损跟踪职责

现有资产分析

配置系统（已存在）

- config/loader.py：load_strategy_config(strategy_name, symbol)动态加载配置

- config/env_loader.py：加载环境变量和API密钥

- config/settings.yaml：全局设置

- config/smc/：已有4个币种配置文件示例

- config/orderflow/：空目录（目标位置）

代码架构

1. 三号引擎核心：engines/engine_3_orderflow/strategy.py（Engine3Commander）

2. 订单流策略：src/strategy/orderflow.py（OrderFlowMath状态机）

3. 交易执行：src/execution/trader.py（OKXTrader含智能止损）

4. 数据层：src/data_feed/okx_stream.py（OKXTickStreamer）

5. 二号引擎集成：engines/engine_2_smc/strategy.py（MicroSMCRadar）

import threading

from typing import Optional, Dict, Any

class MarketContext:

     """线程安全的市场情报池"""

     def __init__(self):

         self._lock = threading.RLock()

         self._reset()



     def _reset(self):

         """重置所有状态"""

         self.of_wall_price = 0.0  # 订单流发现的隐形墙价格

         self.of_squeeze_flag = False  # 订单流拉响的空头挤压警报

         self.smc_support = 0.0  # SMC发现的支撑位

         self.smc_resistance = 0.0  # SMC发现的阻力位

         self.position_info = None  # 当前持仓信息

         self.last_signal = None  # 最后生成的信号

         self.tick_info = {}  # 最新的tick信息



     def update_of_wall(self, price: float):

         with self._lock:

             self.of_wall_price = price



     def update_of_squeeze(self, flag: bool):

         with self._lock:

             self.of_squeeze_flag = flag



     def update_smc_levels(self, support: float, resistance: float):

         with self._lock:

             self.smc_support = support

             self.smc_resistance = resistance



     def update_position(self, position_info: Dict[str, Any]):

         with self._lock:

             self.position_info = position_info



     def update_tick(self, tick: Dict[str, Any]):

         with self._lock:

             self.tick_info = tick



     def update_signal(self, signal: Dict[str, Any]):

         with self._lock:

             self.last_signal = signal



     def get_snapshot(self) -> Dict[str, Any]:

         """获取当前上下文的快照（用于日志和诊断）"""

         with self._lock:

             return {

                 "of_wall_price": self.of_wall_price,

                 "of_squeeze_flag": self.of_squeeze_flag,

                 "smc_support": self.smc_support,

                 "smc_resistance": self.smc_resistance,

                 "has_position": self.position_info is not None,

                 "last_signal": self.last_signal,

                 "tick_price": self.tick_info.get('price', 0.0) if self.tick_info else 0.0

             }
